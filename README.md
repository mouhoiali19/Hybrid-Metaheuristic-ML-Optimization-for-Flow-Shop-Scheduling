Hybrid Metaheuristic-ML Optimization for Flow Shop Scheduling

• Built a Q-Learning guided Genetic Algorithm (RL-GA) to solve the PFSP, where the RL agent dynamically
selects selection method and mutation rate based on population fitness and entropy (9 states × 27 actions).
• Benchmarked against NEH, Tabu Search and Simulated Annealing on Taillard instances (20–100 jobs), achieving up to 4% makespan improvement over standard GA; results written as an IEEE article.
• Built a second method :  ML-VNS approach to solve the PFSP with also a dynamic RL agent.
